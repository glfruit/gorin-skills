# 教材通用审核清单

> 审核前选择对应出版社要求文件，逐项检查。

## 一、结构检查

- [ ] 书名、作者、ISBN 信息完整
- [ ] 前言、内容简介、目录齐全
- [ ] 参考文献数量 ≥ 5 条，含近两年文献
- [ ] 层级结构统一（不超过四级标题）
- [ ] 全书格式一致（字体、字号、行距）

## 二、内容检查

- [ ] 无 TBD/待定/TODO 等占位符
- [ ] 同一概念全书术语统一
- [ ] 代码已重新运行验证
- [ ] 图序、图题、表序、表题齐全且连续
- [ ] 公式编号连续、格式统一

## 三、图片检查

- [ ] 所有图片 ≥ 300dpi
- [ ] 无地图、领导人、公众人物照片
- [ ] 图片居中，图题在图下方
- [ ] 图片引用与正文对应

## 四、中英文检查

- [ ] 专有名词首次出现给出英文全称+缩写
- [ ] 中英文之间有空格
- [ ] 英文缩写使用统一

## 五、思政检查

- [ ] 素质目标中有思政元素
- [ ] 理论内容中有自然融入的思政案例
- [ ] 思政融入不生硬

## 六、AI 融入检查（如适用）

- [ ] AI 融入方式明确（拓展/辅助/替换）
- [ ] AI 相关内容准确、不过度宣传
- [ ] AI 助学板块内容为专业拓展知识
