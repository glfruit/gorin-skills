# 大纲模板列结构

## 教材大纲 (textbook) — 13列

| 列 | 表头 | 必填 | 说明 |
|----|------|------|------|
| A | 项目名称 | ✓ | 如"项目一 数控车削篇" |
| B | 任务名称 | ✓ | 如"任务一 数控车削基础认知" |
| C | 模块/板块 | ✓ | 如"任务目标""相关知识" |
| D | 单元名称 | ✓ | 如"一、认识数控车床" |
| E | 知识点/技能点 | ✓ | 如"数控车床型号、组成结构" |
| F | 主要内容概要 | ✗ | 教学内容简要描述 |
| G | 思政元素 | ✗ | 课程思政融入点 |
| H | 需补充资源 | ✗ | 缺失资源清单 |
| I | 已有资源名称 | ✗ | 已有素材 |
| J | 资源类型 | ✗ | 课件/微课/视频等 |
| K | 数量 | ✗ | 资源数量 |
| L | 负责人 | ✗ | 制作责任人 |
| M | 备注 | ✗ | 补充说明 |

## 课程大纲 (course) — 10列

| 列 | 表头 | 必填 | 说明 |
|----|------|------|------|
| A | 项目/模块名称 | ✓ | 教学项目或模块 |
| B | 任务/单元名称 | ✓ | 教学任务或单元 |
| C | 学时（理论） | ✓ | 数字 |
| D | 学时（实践） | ✓ | 数字 |
| E | 学时（总计） | ✓ | 公式 =C+D |
| F | 教学目标 | ✓ | 知识/技能/素质目标 |
| G | 重点难点 | ✗ | 重点 + 难点 |
| H | 教学方法 | ✗ | 讲授/实操/案例等 |
| I | 考核方式与权重 | ✗ | 过程+结果考核 |
| J | 备注 | ✗ | 补充说明 |

## 培训大纲 (training) — 10列

| 列 | 表头 | 必填 | 说明 |
|----|------|------|------|
| A | 培训模块名称 | ✓ | 培训模块 |
| B | 培训活动名称 | ✓ | 具体活动 |
| C | 时长（分钟） | ✓ | 数字 |
| D | 培训对象/人数 | ✓ | 参训人员 |
| E | 培训方式 | ✓ | 讲授/实操/研讨等 |
| F | 活动内容描述 | ✓ | 活动详细内容 |
| G | 产出物 | ✗ | 培训产出 |
| H | 评估标准 | ✗ | 合格标准 |
| I | 所需资源/设备 | ✗ | 设备/材料清单 |
| J | 备注 | ✗ | 补充说明 |
