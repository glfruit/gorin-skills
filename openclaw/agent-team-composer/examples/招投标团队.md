# 示例：招投标团队

## 输入

```text
帮我组建一个招投标团队，致力于招投标全流程支持。
团队包括：
- 申报主管：负责统筹协调
- 信息雷达：负责信息搜集
- 材料撰写师：负责材料写作
协作模式：并行
告警通道：Telegram
日报输出：Discord
```

## 预期输出

生成目录：

```text
out/招投标团队/
├── SOUL.md
├── AGENTS.md
├── CollaborationRules.md
├── Contacts.md
└── agent-config.yaml
```

## 结果特征

- `workflow` 应识别为 `并行`
- 平台应包含 `telegram` 和 `discord`
- 角色至少包含：
  - `申报主管`
  - `信息雷达`
  - `材料撰写师`

## 运行命令

```bash
npx ts-node scripts/main.ts "帮我组建一个招投标团队，致力于招投标全流程支持。
团队包括：
- 申报主管：负责统筹协调
- 信息雷达：负责信息搜集
- 材料撰写师：负责材料写作
协作模式：并行
告警通道：Telegram
日报输出：Discord" --output-dir ./out
```
