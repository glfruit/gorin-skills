# 示例：内容创作团队

## 输入

```text
帮我组建一个内容创作团队，负责选题、写作、审校和分发。
团队包括：
- 主编：负责选题和最终把关
- 写作者：负责初稿撰写
- 审校员：负责事实核查和语言润色
- 分发运营：负责 Telegram、微信和 X 的发布
协作模式：hybrid
主沟通：Telegram
同步渠道：WeChat
```

## 预期输出

- 团队目录名应为 `内容创作团队`
- 平台应至少识别：
  - `telegram`
  - `wechat`
- 协作模式应识别为 `hybrid`
- `agent-config.yaml` 中应包含 4 个角色定义

## 用途

这个示例适合验证：

- 多角色自然语言提取
- 多平台 IM 识别
- `hybrid` 工作流模式
- Markdown + YAML 混合输出的可读性
