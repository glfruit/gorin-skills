# User Guide — Idea Creator

## Quick Start

### Basic Usage
Simply type:
```
想法 [your idea]
```

Example:
```
想法 AI should assist humans, not replace them
```

That's it! The system will:
1. Create a note in `Zettels/1-Fleeting/`
2. Search for related notes
3. Add relationship links
4. Show you the results

## Understanding Relationships

When you create an idea, the system finds related notes and classifies them:

### ✅ Supports
Notes that reinforce or agree with your idea.

Example:
- Your idea: "AI should assist humans"
- Supporting note: "Human autonomy is most important"

### ❌ Contradicts
Notes that oppose or challenge your idea.

Example:
- Your idea: "AI should assist humans"
- Contradicting note: "AI will fully replace humans"

### 🔍 Related
Notes on similar topics but without clear stance.

Example:
- Your idea: "AI should assist humans"
- Related note: "How to use AI tools effectively"

### 🌱 Extends
Notes that your idea builds upon or develops further.

Example:
- Your idea: "AI assists + human judges"
- Extended from: "AI-assisted programming"

### 💭 Cluster
Other ideas on similar topics.

Example:
- Your idea: "AI-assisted programming"
- Cluster: "AI-assisted writing", "AI-assisted design"

## Advanced Usage

### Adding Tags
```
想法 AI in education
tags: ai, education, future
content: AI will transform how we learn...
```

### Adding Context
```
想法 Reduce meetings, increase async
context: After today's 3-hour meeting
content: Synchronous meetings are productivity killers...
```

### Developing Ideas

**From Fleeting to Permanent**:
1. Review idea weekly
2. Add more evidence and reasoning
3. Link to more sources
4. Move to `Zettels/3-Permanent/`

**Creating Idea Clusters**:
Group similar ideas:
```
AI-assisted Programming
├── Idea: AI code completion
├── Idea: AI code review
└── Idea: AI pair programming
```

## Best Practices

### 1. Capture Immediately
Don't wait! Ideas fade quickly.

✅ Good: 
```
想法 [in the shower] Water flow optimization...
```

❌ Bad:
"I'll remember this idea later" [forgotten]

### 2. Be Concise
One clear idea per note.

✅ Good:
```
想法 Async communication > meetings for deep work
```

❌ Bad:
```
想法 About work, and life, and how we communicate, 
and meetings, and async, and productivity, and...
```

### 3. Review Weekly
Every Friday:
1. Browse `Zettels/1-Fleeting/`
2. Select promising ideas
3. Develop into permanent notes
4. Archive or delete weak ideas

### 4. Build Connections
Don't just collect ideas—connect them!

- Link supporting ideas
- Note contradictions
- Form clusters
- Create synthesis

## Troubleshooting

### No related notes found?
- Your vault might be new
- Keep adding notes
- Relationships will emerge

### Wrong relationships?
- Edit the note manually
- Change relationship types
- Add missing links

### Too many unrelated notes?
- Refine your keywords
- Add more specific tags
- Manually curate links

## Tips

1. **Use consistently**: Make it a habit
2. **Trust the process**: Connections emerge over time
3. **Edit freely**: Auto-links are starting points
4. **Have fun**: Ideas are playful!
