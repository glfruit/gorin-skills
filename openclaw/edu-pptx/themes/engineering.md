# Engineering — 工科/制造/机械

Forest Canopy 变体，深绿工业风。适合数控、机械、焊接、汽车等课程。

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `2C5F2D` | 主色，标题页背景、重点色块 |
| Secondary | `97BC62` | 辅色，章节导航、图标背景 |
| Accent | `F5F5F5` | 点缀，浅背景、分隔线 |
| Dark | `1A3C2A` | 深色，封面/结束页背景、页脚 |

## Typography

| Element | Font | Size |
|---------|------|------|
| 标题 | Georgia Bold | 36-44pt |
| 章节标题 | Georgia Bold | 24-28pt |
| 正文 | Calibri | 14-16pt |
| 代码 | Consolas | 12-13pt |

## Layout Rules

- 封面/结束页：Dark 背景 + 白色大字
- 内容页：Accent 背景 + Primary 色块装饰
- 表格表头：Primary 背景 + 白色文字
- 代码块：`36454F` 背景 + `97BC62` 关键字高亮
- 侧边栏：Primary 左侧色带 0.08"
