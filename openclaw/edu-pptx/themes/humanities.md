# Humanities — 人文/传统文化

Desert Rose 变体，水墨暖色调。适合语文、历史、传统文化、艺术等课程。

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `B85042` | 主色，赤陶/朱砂红 |
| Secondary | `E7E8D1` | 辅色，宣纸米色背景 |
| Accent | `A7BEAE` | 点缀，青瓷绿 |
| Dark | `4A3728` | 深色，封面/结束页，墨色 |

## Typography

| Element | Font | Size |
|---------|------|------|
| 标题 | Palatino Bold | 36-44pt |
| 章节标题 | Palatino Bold | 24-28pt |
| 正文 | Garamond | 14-16pt |
| 引用 | Palatino Italic | 14-16pt |

## Layout Rules

- 封面/结束页：Dark 背景 + 浅色大字
- 内容页：Secondary 背景，留白充足
- 引用页：居中大号引文 + Accent 装饰线
- 章节过渡页：大幅留白 + Primary 色块
- 避免密集排布，注重呼吸感
