# Safety — 安全/应急教育

Ocean Depths 变体，深蓝专业风。适合安全教育、消防、应急救护等课程。

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `1A2332` | 主色，深海蓝 |
| Secondary | `2D8B8B` | 辅色，青色警示 |
| Accent | `F1FAEE` | 点缀，奶油白背景 |
| Danger | `E63946` | 警示色，禁止/危险标识 |

## Typography

| Element | Font | Size |
|---------|------|------|
| 标题 | Calibri Bold | 36-44pt |
| 章节标题 | Calibri Bold | 24-28pt |
| 正文 | Calibri | 14-16pt |
| 警示文字 | Calibri Bold | 16-18pt Danger 色 |

## Layout Rules

- 封面/结束页：Primary 背景 + 白色大字
- 警示页：Danger 背景 + 白色文字 + 大号警示图标
- 步骤页：数字编号 + Secondary 色块
- 检查清单：✓/✗ 图标 + 左对齐
- 规程页：编号列表 + 必要时 Danger 高亮重点
