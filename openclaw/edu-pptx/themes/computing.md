# Computing — 计算机/IT/开发

Tech Innovation 变体，深蓝科技风。适合编程、网络、数据库、人工智能等课程。

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `065A82` | 主色，标题页、导航栏 |
| Secondary | `1C7293` | 辅色，图标、链接 |
| Accent | `E0F7FA` | 点缀，代码背景、信息框 |
| Dark | `21295C` | 深色，封面/结束页 |

## Typography

| Element | Font | Size |
|---------|------|------|
| 标题 | Calibri Bold | 36-44pt |
| 章节标题 | Calibri Bold | 24-28pt |
| 正文 | Calibri | 14-16pt |
| 代码 | Consolas | 12-13pt |

## Layout Rules

- 封面/结束页：Dark 背景 + 白色大字
- 代码页整页深灰 `2D2D2D` 背景 + Consolas
- 表格表头：Primary 背景 + 白色文字
- 流程图：Primary 节点 + Secondary 连线
- 每页至少一个视觉元素（代码块/表格/流程图）
