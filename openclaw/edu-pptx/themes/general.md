# General — 通识/职业素养

Modern Minimalist 变体，灰白简约。适合职业素养、创新创业、心理健康等课程。

## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Primary | `36454F` | 主色，深灰 |
| Secondary | `708090` | 辅色，石板灰 |
| Accent | `028090` | 点缀，蓝绿色强调 |
| Dark | `212121` | 深色，封面/结束页 |

## Typography

| Element | Font | Size |
|---------|------|------|
| 标题 | Calibri Bold | 36-44pt |
| 章节标题 | Calibri Bold | 24-28pt |
| 正文 | Calibri | 14-16pt |
| 引用 | Calibri Italic | 14-16pt |

## Layout Rules

- 封面/结束页：Dark 背景 + 白色大字
- 内容页：白色背景，Primary 文字
- 要点卡片：Accent 左边框 0.06"
- 图标：Accent 色圆形背景 + 白色图标
- 表格表头：Primary 背景 + 白色文字
