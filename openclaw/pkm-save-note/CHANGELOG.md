## [1.0.0] - 2025-02-22

### Added
- Initial skill creation
- Agent-usable interface for saving notes
- Smart PARA classification (Projects/Areas/Zettels)
- Auto-tagging based on content analysis
- Related notes discovery
- Multiple templates (fleeting, zettel, meeting, plan, review)
- Dry-run mode for testing
- Support for explicit project/area targeting

### Design Goals
- **Agent Interoperability**: Any agent can call this to save structured notes
- **Zero-config Defaults**: Works out of the box with sensible defaults
- **Extensible**: Easy to add new templates and classification rules
