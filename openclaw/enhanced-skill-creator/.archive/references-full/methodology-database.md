# Methodology Database

A curated collection of expert methodologies organized by domain. Use these as starting points when researching golden cases in Step 2 of the skill creation workflow.

## 3-Layer Discovery Strategy

When searching for methodologies in a new domain, work through these layers:

### Layer 1: Established Frameworks (fastest)
Named methodologies with books, courses, or widespread adoption. Search: `"{domain} framework"`, `"{domain} methodology"`.

### Layer 2: Practitioner Patterns (most actionable)
Blog posts, conference talks, and open-source tools by domain experts. Search: `"{domain} best practices" site:github.com`, `"{domain} expert workflow"`.

### Layer 3: Academic/Theoretical (deepest)
Research papers and formal models. Use only when Layer 1-2 don't explain *why* something works. Search: `"{domain} systematic review"`, `"{domain} meta-analysis"`.

## How to Use This Database

1. **Find your domain** in the tables above
2. **Pick 1-2 frameworks** most relevant to your skill's task
3. **Search for the source material** — read the actual book/article, not just summaries
4. **Extract specific techniques** — not "use JTBD" but "identify the functional, emotional, and social jobs the user is trying to accomplish"
5. **Combine with practitioner patterns** from Layer 2 searches for a complete picture
