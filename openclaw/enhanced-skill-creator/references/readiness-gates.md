# Readiness Gates

| Level | Requirements | Label |
|-------|-------------|-------|
| 0 | SKILL.md with frontmatter | `scaffold` |
| 1 | + real content (no placeholders) | `mvp` |
| 2 | + validate-skill passes + golden case test | `candidate` |
| 3 | + internal acceptance + integration proven | `production-ready` |
| 4 | + 3+ real-world uses + governance score ≥ 80 | `integrated` |

Never skip levels. Never label `integrated` without proven integration.
