# Governance Model

## Maturity Tiers
- **scaffold** → **candidate** → **published** → **integrated**
- Promotion requires evidence (golden cases, acceptance results, governance score)

## Review Cadence
| Tier | Cadence | Action |
|------|---------|--------|
| scaffold | On-demand | Creator self-review |
| candidate | Monthly | Peer review |
| published | Quarterly | Quality audit |
| integrated | Quarterly | Usage metrics + quality audit |

## .skill-meta.json Required Fields
`lifecycle`, `maturity_score`, `last_reviewed`, `review_cadence`, `promotion_evidence`
