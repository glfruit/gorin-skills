# Quality Standards (v4 — 6 Dimensions, 120 Points)

Full rubric in `.archive/references-full/quality-standards.md`.

| Dimension | Max | Key Criteria |
|-----------|-----|-------------|
| Content Quality | 25 | Real examples, no placeholders, golden case alignment |
| Trigger Precision | 20 | Specific triggers, negative cases, overlap-free |
| Error Handling | 20 | All failure modes covered, recovery paths defined |
| Context Efficiency | 20 | Within token budget, progressive disclosure |
| Governance | 20 | .skill-meta.json, review cadence, promotion evidence |
| Integration | 15 | Internal acceptance passed, real happy-path proven |

**Minimum for candidate**: 70/120. **Minimum for production**: 90/120.
